I love making things in my free time. Here are some creative coding experiments, crafting projects, games, apps, and more.

Rainbow
vibecoded with Gemini
A lightweight tool for multilabel span annotation. It's just a single HTML file, so compared to some of the professional tools out there it's great for small workloads or for task prototyping.
Terraincraft

Miniature Keyboard

Interactive Vector Cloud
A python -> processing pipeline.

Infinite Letterboxing
Python to generate games; Processing UI to play them.
In college I loved playing Letterboxed (from the NYTimes) with my friends. They only release one game a day, so I built this app to let us keep playing forever.
Simmer (formerly "Foodie")
YCombinator 2019.
Some of my friends in college built an app to help people find the best dish at any restaurant. I joined in to help with the ML pipeline, and built a trend-tracking tool for restaurants. We got into YC, but the app is defunct now.
D&D Token Crafting
Built with laser cut acrylic and wood, vinyl, wire, imagination.
In college, I played Dungeons & Dragons with my friends and loved to design worlds, characters, and tabletop objects. At UChicago's Fab(ulous) Lab, we made artistic player tokens and laser-cut wooden architectural maps.
